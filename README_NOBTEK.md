# Nobtek | نوبتک

نوبتک یک پلتفرم فارسی رزرو آنلاین است که برای هر کسب‌وکار یک صفحه اختصاصی رزرو می‌سازد. این نسخه با Django ساخته شده و شامل پنل ادمین، پنل صاحب کسب‌وکار، خدمات قابل رزرو، تقویم کاری، رزرو عمومی، پرداخت آزمایشی، پلن‌های اشتراکی، سئو پایه و رابط کاربری RTL مدرن است.

## ساختار اپ‌ها

- `pages`: صفحه اصلی، تعرفه‌ها، دسته‌بندی‌ها، تماس، FAQ، تنظیمات سایت، robots و sitemap.
- `accounts`: ثبت‌نام، ورود، داشبورد صاحب کسب‌وکار، مدیریت خدمات، ساعت کاری، رزروها و اشتراک.
- `businesses`: دسته‌بندی، پروفایل کسب‌وکار، خدمات، کارکنان، ساعت کاری و زمان‌های بسته.
- `bookings`: ثبت رزرو عمومی، وضعیت رزرو، پرداخت آزمایشی و تراکنش‌ها.
- `plans`: پلن رایگان، نقره‌ای، طلایی، سازمانی و محدودیت‌های هر پلن.

## اجرای محلی

```bash
cd Nobtek
python -m venv .venv
# Windows
.venv\Scripts\activate
# Linux/macOS
source .venv/bin/activate
pip install -r requirements.txt
python manage.py migrate
python manage.py seed_nobtek
python manage.py runserver
```

سپس باز کنید:

- سایت: `http://127.0.0.1:8000/`
- ادمین: `http://127.0.0.1:8000/admin/`

## نکات امنیت و پروداکشن

قبل از انتشار واقعی:

1. مقدار `DJANGO_SECRET_KEY` را در محیط سرور تغییر بده.
2. `DJANGO_DEBUG=False` قرار بده.
3. `DJANGO_ALLOWED_HOSTS` را روی دامنه واقعی تنظیم کن.
4. برای HTTPS این موارد را فعال کن: `DJANGO_COOKIE_SECURE=True`، `DJANGO_SECURE_SSL_REDIRECT=True` و HSTS.
5. برای پرداخت واقعی باید درگاه بانکی به مدل `PaymentTransaction` وصل شود.
6. برای ارسال پیامک واقعی باید سرویس پیامک به نقاط ثبت/تغییر رزرو اضافه شود.
7. برای تولید استاتیک‌ها اجرا کن: `python manage.py collectstatic`.

## منطق رزرو

موتور زمان‌بندی در `businesses.models.get_available_slots` قرار دارد و این موارد را چک می‌کند:

- ساعت کاری کسب‌وکار
- زمان استراحت
- زمان‌های بسته‌شده دستی
- رزروهای قبلی و تداخل زمانی
- ظرفیت هر خدمت
- آینده بودن زمان رزرو
- مدت‌زمان خدمت و فاصله بعد از رزرو
